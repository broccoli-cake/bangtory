## 🧩 프로젝트 개요

**방토리**는 다음과 같은 문제를 해결하고자 합니다:

- 룸메이트 간의 **공용 일정 조율**
- **실시간 피드백 및 소통**
- **예약 및 공유 공간 사용 관리**
- **가사 분담과 같은 생활 관리**

> 단순한 공동생활이 아닌, **함께 살아가는 경험을 개선**하는 데 중점을 둡니다.
